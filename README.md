# AI Content Repurposer

This is a simple AI-powered tool that takes a YouTube video link and extracts the transcript.

The goal of this project is to later convert long-form video content into multiple social media posts automatically.

## Features (Current)
- Paste a YouTube video link
- Extract and display the video transcript

## Tech Stack
- Python
- Streamlit
- youtube-transcript-api
